# 🌐 Portafolio Web - Natalia Tavera

## Descripción

Este repositorio contiene el desarrollo de mi portafolio web personal, creado para presentar mi perfil profesional, habilidades, experiencia y proyectos en un sitio web moderno, responsivo y de fácil navegación.

El objetivo de este proyecto es fortalecer mis conocimientos en desarrollo web y contar con una página profesional que sirva como carta de presentación para oportunidades laborales y proyectos independientes.

## Instalación

Clona el repositorio:

```bash
git clone git@github.com:nataliatavera758/web_project_portfolio_es.git
```

Ingresa al proyecto:

```bash
cd web_project_portfolio_es
```

Abre el archivo `index.html` en tu navegador o utiliza la extensión **Live Server** en Visual Studio Code.
